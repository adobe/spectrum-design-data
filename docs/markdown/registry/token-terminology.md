| ID | Label | Description | Aliases |
| --- | --- | --- | --- |
| component | Component | In token naming, refers to the UI component the token is designed for | - |
| object | Object | In token naming, refers to the specific part or element being styled | - |
| scale | Scale | In token naming, refers to the size or intensity level | - |
| property | Property | In token naming, refers to the CSS/style property being defined | - |
| edge | Edge | In token naming, refers to the outer boundary of a component | - |
| visual | Visual | In token naming, refers to the visible graphic element of a component | - |
| structure | Structure | A reusable visual pattern or object category with shared styling that occurs across multiple component types | - |
| base | Base | Basic UI elements and visual structures combined to create proportionally sized content or components | - |
| list | List | A vertically or horizontally oriented collection of items with varying complexity | - |
| table | Table | A grid structure arranging columns and rows of content | - |
| container | Container | A visible or invisible structure that houses custom content and components with consistent styling | - |
| group | Group | A collection of similar or related UI elements, inputs, or actions grouped together | - |
| drop-target | Drop Target | Visual indication of a UI region that can accept drop operations | - |
| focus-ring | Focus Ring | Visual indication of interactive keyboard focus on any UI element | - |
| accessory | Accessory | Component-like structures used within basic components as secondary content or actions | - |
| banner | Banner | Horizontally-oriented, contained structure for drawing attention to non-permanent information or actions | - |
| text | Text | Text content within a component or structure | - |
| substructure | Substructure | A structural element that exists within or as part of a larger structure | - |
| item | Item | An individual element of a collection, regardless of collection type | - |
| padding | Padding | The internal spacing from the edges of a component's boundary to its contents | - |
| gap | Gap | The internal spacing between child elements in a component, layout, or pattern | - |
| orientation | Orientation | The direction or order of structures and elements within a component or pattern | - |
| shape | Shape | Describes the geometric characteristics of a component's style attributes | - |
| uniform | Uniform | Signifies that the shape of a component's style attributes is equal in all directions | - |
| position | Position | The placement of an element relative to its container or viewport | - |
| affixed | Affixed | The position of an element fixed relative to its container or the viewport | - |
| size | Size | Relative t-shirt-size terms used to create relationships and patterns across tokens | - |
