| Property    | Type   | Values               | Default    | Required | Description |
| ----------- | ------ | -------------------- | ---------- | -------- | ----------- |
| size        | string | s, m, l              | s          | No       | -           |
| orientation | string | horizontal, vertical | horizontal | No       | -           |
