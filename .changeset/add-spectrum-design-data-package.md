Initial release of `@adobe/spectrum-design-data` — Spectrum design tokens in cascade format.

This package is the new canonical source of truth for `@adobe/spectrum-tokens`. It holds 4 166 cascade tokens (8 files) with structured name objects derived from the full token taxonomy. `packages/tokens/src/` is regenerated from this package via `moon run design-data:legacy-output`.
